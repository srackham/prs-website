Contractor Pay Advice Templates

In addition to a default printed contractor pay advice layout you can fully customise the printed pay advice by creating a Pay Advice Template document. A Pay Advice Template is a Microsoft Word document containing bookmarks that mark where the PRS is to insert pay advice data. You need to have Microsoft Word installed to use the template feature.

 

To create a Contractor Pay Advice Template press the Edit Pay Advice Template... button on the Timesheet Options dialog (Tools->Timesheets->Options... menu command). If the template does not exists it will be created using the default PRS Pay Advice Template.

 

The contractor pay advice template file must resides in the PRS Shared Data folder and is named payadvice_template.doc.
