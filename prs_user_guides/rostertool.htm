Roster Tool

The Roster Tool provides for the management and processing of all roster entries.

 

Open the Roster Tool from the PRS Tools menu and use the Roster Commands to manage the roster entries.

 

TIPS:

•Click the mouse on underlined column headings to sort Roster Tool entries.
•Use the Goto buttons on the roster forms to view and edit candidate details and pay rates.
•Use the Calendar command to select roster entries for a week or a day.
•Use the Send Messages command to Email or SMS the personnel in the current roster selection.
•Use the Edit command to bulk update the selected roster entries.
•You can resize or maximise the Roster Tool window to full-screen, this is handy when working with large number of roster entries.
