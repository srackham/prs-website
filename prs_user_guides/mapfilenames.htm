Map File Names

Mapped files are named like: <id>_MappedLocations.csv, <id>_MappedOccupations.csv and <id>_MappedIndustries.csv where <id> is the vacancy file Format ID:

 

Format ID

Format

CSV

Comma Separated Variable

XLS

Microsoft Excel

HTML

Web Pages

XML

XML

 

For example, if the Export Folder is set to \\QUARK\public\Export\ then the SEEK industries Map File would be named \\QUARK\public\Export\MapFiles\XLS_MappedIndusries.csv. The first values in each line will contain the PRS value and the second the exported mapped value.
