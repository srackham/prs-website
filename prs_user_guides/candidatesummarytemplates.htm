Candidate Summary Templates

Candidate Summary information can be viewed, edited and printed by pressing the Candidate Summary button on the Personnel form.

 

Candidate Summary templates allow printed candidate summaries to be fully customised. A Candidate Summary Template is a Microsoft Word document containing bookmarks that mark where the PRS will insert candidate summary fields.

 

Candidate Summary templates reside in the PRS Shared Data folder, if you're unsure where this is check File Locations using the PRS Tools->Options... menu command.

 

By default the PRS uses the built-in Candidate Summary report, rather than a Candidate Summary template, to print candidate summaries. You can create a Candidate Summary template by pressing the Edit Report Template... button on the Personnel Candidate Summary form.

 

The PRS assumes file names starting with Candidate Summary in the Shared Data folder are Candidate Summary templates. You can have more than one template, in which case you will be asked to make a selection when you print a candidate summary.
