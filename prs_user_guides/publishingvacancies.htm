Publishing Vacancies

Press the Publish button on the Vacancies form to enter vacancy publication details.

 

There can be up to six user definable Publication options. Options 1 to 4 are defined in the example above (SEEK, Featured, TradeMe, Featured). The Administration Guide explains how to enable and configure these options.

 

Published vacancies can be exported to HTML Web pages plus a number of other file formats using the Tools->Vacancy Publication menu command.

 

Published vacancies can be individually selected for publication to Web pages, vacancy files, or both.

 

Since recruitment company and consultant details are included in exported vacancy data it is important that they are entered and accurate. Namely:

 

•Recruitment company Phone and Fax numbers plus Email and Web Site addresses in the company's Site record.
•Consultant Name and Email address in the consultant's Personnel record.

 

Administrative privilege is required to enable vacancy publication.
