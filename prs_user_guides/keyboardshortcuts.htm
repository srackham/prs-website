Keyboard Shortcuts

Key Combination

Function

ALT+G

Go to record (note 1)

ALT+F

Find records (note 1)

ALT+N

Create new record (note 1)

ALT+D

Delete current record (note 1)

ALT+S

Show all records (note 1)

F4 or CTRL+L

Lock and Save / Unlock current form (note 1).

F5

Open the Goto List. If the Goto List is open then pressing F5 will add the current Personnel, Sites or Vacancies form record to the Goto List.

F7

Spell check.

F8

Open and close Favorites window

F9

View Document Search Tool

F12

Size to Full Screen.

ESC

Undo edit on current field.

ESC ESC

Undo edit on current record.

TAB

Go to next field on the form.

SHIFT+ENTER

Save current record.

CTRL+F

Find a record based on the current field (note 3).

CTRL+S

Skills Search command.

PAGE UP

Move to previous record on form.

PAGE DOWN

Move to next record on form.

CTRL+HOME

Move to first record on form.

CTRL+END

Move to last record on form.

CTRL+SHIFT+F

Switch form / datasheet view on current form (note 1).

CTRL+R

Update form drop-down list data.

CTRL+SHIFT+A

Sort records ascending.

CTRL+SHIFT+D

Sort records descending.

CTRL+SHIFT+I

View Invoices form.

CTRL+SHIFT+P

View Personnel form.

CTRL+SHIFT+S

View Sites form.

CTRL+SHIFT+T

Go to a Timesheet.

CTRL+SHIFT+V

View Vacancies form.

CTRL+SHIFT+J

Go to the Current Rostered Vacancy.

CTRL+ ;

Enter today's date in the current field.

CTRL+SHIFT+ ;

Enter the time in the current field.

CTRL+SHIFT+F2

Zoom (view the current field in a text dialog).

Notes:

1.Personnel, Sites, Vacancies, Invoices forms.
2.Personnel, Sites forms.
3.Unlock Personnel, Sites, Vacancies, Invoices forms to enable Replace option.
