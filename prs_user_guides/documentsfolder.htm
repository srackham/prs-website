Documents Folder

The shared Documents Folder specifies the location of company recruitment related documents (typically CVs and resumes).

 

The shared Documents Folder is the location of shared linked documents and can be set globally for all PRS Clients from the General page of the PRSMon Tools->Options... dialog. Once the Documents folder option has been set using PRSMon users will be prevented from changing the Documents folder (using the PRS Tools->Options... menu command).

 

The shared Documents Folder location must be accessible and writeable from all PRS Client PCs.

 

See also Managing Shared Data Locations.
