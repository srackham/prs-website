Why do printed reports overflow the right hand margin?

Probably because your printer is not set to print to A4 paper (PRS reports are optimised for A4 size paper). Check your printer properties from the Settings sub-menu of the Windows Start Menu and make sure the default paper size is set to A4. If you change the paper setting from the PRS print dialog then the change will only affect the current report, change the Printer Settings to effect a permanent change.
