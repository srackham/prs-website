PRS Version 7

Version 8.2.2 is the latest PRS version 7 release.

See the PRS News page for details of the latest PRS release.

What’s New in Release 7

This release introduces two significant new features:

  1. A new flexible record based notes system replaces the old Personnel, Sites, Vacancies and Invoices note fields.

  2. You can set any person, site, vacancy or invoice as a Favorite for instant access via the floating Favorites form (just press the F8 key to bring up the Favorites form).

Note
 The move from PRS version 6 to 7 requires a database upgrade and migration (handled by the PRSMon database administration program). For installation and migration instructions see the Updates page.

Revamped and Enhanced Notes System

  • Individual notes can be 60KB long — previously all notes attached to a record could not total more then 60KB.

  • Each note has is assigned a Note Type. Note Types are user definable using the PRS Tools→Setup→Note Types menu command.

  • Notes can now be viewed and reported by Note type and date (using the new Notes Tool). For example, you could now view and report all consultant Client visits notes in the last month.

  • Notes form a valuable audit history since they can only be deleted by a supervisor and can be marked read-only to inhibit further changes (all system generated notes are marked read-only by default).

  • The new Notes system replaces the old PRS Transaction Log.

More information about the new Notes system can be found in the Notes section of the PRS Essential Reference.

Favorites

To add an item to your personalised list of Favorites click on the Favorite sidebar menu item on the Personnel, Invoice, Sites or Vacancies forms.

Now press the F8 key to bring up a floating, sizable lists of your favorites.

More information about the new Favorites system can be found in the Favorites section of the PRS Essential Reference.

Version 7 User Interface changes

  • Notes entry on Personnel, Invoice, Sites and Vacancies forms is unchanged except that you Ctrl+click on existing notes to edit them.

  • A Favorite sidebar menu command has been added Personnel, Invoice, Sites and Vacancies forms to add the record to your list of favorites.

The following new PRS menu commands have been added:

Tools→Notes

Using the Notes Tool you can find, view and print selected notes.

Reports→Notes

Use this command to select and print Notes reports.

Tools→Setup→Notes Types

Use this command to add or change Note Types.

View→Favorites

Open the floating Favorites form (shortcut F8).
