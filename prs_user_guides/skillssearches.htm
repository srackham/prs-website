Skills Searches

Skills can be searched using the following techniques:

 

•To cross match personnel, sites and vacancies use the Match Personnel, Match Sites and Match Vacancies buttons below the skills lists on the Personnel, Sites and Vacancies forms.

 

•To search personnel for skills use the Find button on the Personnel form or the Edit->Skills Search... menu command.

 

•To perform a site or vacancies skill search use the Edit->Skills Search... menu command.

 

•To search for vacancies whose sites have a particular skill use the Find button on the Vacancies form and select the skill from the Site skill drop-down list. You can use this technique, for example, to find all 'preferred supplier' vacancies or to find vacancies within a geographical area (see the Pseudo Skills examples).
