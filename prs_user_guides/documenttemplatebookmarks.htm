Document Template Bookmarks

When a new linked document is created from a Document Template the PRS automatically populates the document with data from the associated linked record. It does this by inserting data fields from the linked parent record into predefined named bookmarks in the template.

 

The set used depends on whether your template is intended for linking to a Personnel, Vacancy or Site record or used in an Invoice Template.

1.Bookmark names starting with User relate the currently logged on PRS user.
2.Invoice bookmarks are for optional PRS Invoice Templates.
3.Vacancy related Invoice Template bookmarks only apply to invoices with a related vacancy ID.
4.If the Fax/Email field on the Personnel form is an Email address then it populates the Email2 Personnel bookmark; if it is a phone number then it populates the Fax Personnel bookmark.
