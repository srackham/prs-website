End-to-end recruitment management and information system

PRS Features

The PRS is a high performance desktop application with a rich user interface and on-premises data (no clunky browser based Cloud application for us). Here are just some of the features that makes the PRS a world beating recruitment solution:

  • The PRS is Internet enabled with Web publication, Email and SMS Texting tools.

  • The PRS is a multiuser relational recruitment management system designed first and foremost to increase consultant productivity and efficiency.

  • The PRS is an interactive recruitment and HR information and management system that manages the entire recruitment process from candidate selection through to client billing.

  • Billing and contract management system.

  • Performance management and CRM system.

  • Built-in reminders messaging system. Resume submission, client interview and job offer management, tracking and reporting.

  • Skills based personnel, site and vacancy searching and matching.

  • Automated skills matching to personnel, sites and vacancies.

  • Extensive personnel and vacancy search criteria.

  • Integrated management of documents (such as CVs, job descriptions and company profiles) which can be linked to personnel, sites and vacancies and opened directly from the PRS.

  • Built-in Skills Scanner automates resume skills extraction.

  • An integrated Document Search Tool that allows Google-like searches of documents for words, groups of words or phrases.

  • The PRS has over two dozen distinct CRM and financial reports.

  • Selection options result in literally hundreds of report combinations.

  • All reports can be exported to Excel, HTML, RTF or text files.

  • Mail-merge selected personnel data directly to the built-in Message Merge tool or using Microsoft Word mail merge.

  • Offline data access for off-site laptop operation.

  • Export vacancies, personnel and reminders directly to Outlook.

  • Built-in vacancy publication management: Vacancies can be published directly to customisable HTML, XML, CSV and Excel file formats.

  • User customisable invoices and tax names.

  • Database audit reports.

  • Extensive use of hyper-linked goto buttons.

  • Built-in personnel, sites and vacancy notes management.

  • Emailing directly from the PRS.

  • Tooltips on buttons and controls.

  • Dial phone number directly from the PRS using Skype or the Windows Dialer program.

  • Consultant commission and billing management.

  • Consultant performance reporting and placement analysis reporting.

  • Client and client site activity reporting.

  • Contracting and temp timesheet and invoice generation for large and small payrolls.

  • The PRS comes with an easy to use administration tool for easy backups, version updates and systems maintenance.
