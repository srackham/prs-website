Export Folder

The folder where exported vacancy files, vacancy web pages and timesheets will be written (set using the Tools->Options... menu command). Files are written to separate subfolders of the Export Folder, depending on the type of export data:

 

Subfolder Name

Contents

ExportedFiles

Palm PC files exported by the Tools->Export... command.

MapFiles

CSV formatted Publication Map Files for translating PRS locations, occupations and industry names to vacancy file specific locations, occupations and industry names.

RosterFiles

Roster data exported by the Roster Tool Process Roster command.

TimesheetFiles

Timesheet files exported using the Timesheet Tools in the Tools->Timesheets sub-menu.

VacancyFiles

Vacancy files exported by the Vacancy Publication Tool (Tools->Vacancy Publication menu command).

WebPages

HTML Web Page files exported by the Vacancy Publication Tool (Tools->Vacancy Publication menu command).

 

If an export command fails to find it's subfolders it will first prompt the user before creating it automatically.
