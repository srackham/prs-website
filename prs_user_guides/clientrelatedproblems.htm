Client Related Problems

If the same problem occurs only on one PC the problem is almost certainly client (local workstation) related.

 

Problem

Cause and Resolution

Unfulfilled client prerequisites

See the Client PC hardware and software requirements.

Client corruption

This may occur if the user switches the PC off at the power switch before closing the PRS. Reinstall the PRS client.

Missing (moved or deleted) files

The user will be unable to successfully start the PRS if any of the client PRS files are inadvertently moved or deleted. The simplest solution is to reinstall the PRS client.

 

Microsoft Office service release (SR) updates not installed.

If you are experiencing problems with PRS features that require Microsoft Office (Outlook for mail and Word for document management) check you have the recommended Microsoft Office service releases installed.

You should also check the Prerequisites section.
