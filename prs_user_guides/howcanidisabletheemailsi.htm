How can I disable the email signature?
•Select Options... from the PRS Tools menu.

 

•Click the Mail tab on the Options dialog.

 

•If the Mail format option is set to HTML:
1.Press the HTML Template... button.
2.Delete the HTML Mail Template text and press the OK button.

 

•If the Mail format is set to Text:
1.Press the Text Signature... button on the Mail Options page.
2.Delete the signature text and press the OK button.
