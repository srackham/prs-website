Skype Phone Calls

To use Skype to make phone and SMS calls from the PRS you need to:

 

1.Configure the PRS to use Skype.
2.Install, configure and test Skype on your PC.

 

TIP: You can place Skype-to-Skype calls from the PRS by putting a Skype name in PRS phone number fields instead of a phone number.
