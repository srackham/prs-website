PRS Support

The PRS includes of wealth of searchable online documentation which can be accessed from the PRS Help menu or online. Make sure you check out the Frequently Asked Questions, a compendium of the most asked PRS support questions.

If you are experiencing operational problems read our article Maintaining Database Reliability and Performance.

Chances are you’ll find the answers to your question in the documentation, if not email us your enquiry using the EMail PRS Support… command from the PRS Help menu (please include a description of the problem in your message and include the wording of any error messages, screenshots are also very helpful).

Alternatively you can email support@prshq.com directly.

PRS Updates

PRS license holders can download the latest PRS updates from the PRS Updates web page. To open the PRS Updates web page select the PRS Updates Web Page menu item in the PRS Help menu. If you need help downloading and installing the latest PRS Client run the 3 minute video Installing Software Updates.

Feedback

Your feedback is important: please send us your suggestions and problem reports using the EMail PRS Support… command in the PRS Help menu,

Alternatively you can manually send your email to support@prshq.com.
