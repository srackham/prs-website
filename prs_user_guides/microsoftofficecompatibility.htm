Microsoft Office Compatibility

The PRS does not require Microsoft Office but the following functionality will not be available unless Office is installed:

 

•Full Mail Tools functionality necessitates Microsoft Outlook.
•The Document Manager Tool (PRS mail merge) uses Microsoft Word.
•Autocorrect functionality necessitates Microsoft Word.

 

The PRS is compatible with Microsoft Office versions 2000, 2002, 2003, 2007, 2010, 2013, Office 365 (32-bit versions).

 

In addition the latest Microsoft Office service packs should be installed. Microsoft Office service releases and installation instructions can be downloaded from the Microsoft website http://www.microsoft.com.
