PRS Demonstration version

The PRS Demonstration version is a self contained fully functional PRS Client and includes an example prepopulated database (licensed to the fictitious company Foobar Recruitment). The Demonstration version is useful for evaluating the PRS and as a training tool.

Note
 Before installing and running the demo you will need to contact us for a demo license activation file.

Installing the PRS Demonstration version

  1. First you need to Download and install the PRS Runtime.

  2. Next download the PRS Demonstration Setup Wizard.

  3. Run the downloaded prs2k_demo_setup_822 Setup Wizard and follow the on-screen instructions.

    Note
    		 Unless you know exactly what you’re doing accept the setup wizard defaults.

  4. Double-click the PRS 2000 Demonstration icon on the Desktop.

  5. When you are prompted for PRS license file select the demo license file that you received via email.

  6. Select any name from the Consultant name drop-down list and press the Logon button.
